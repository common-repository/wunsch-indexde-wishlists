== Description ==

This widget allows you to add your wunsch-index.de wishlist to your blog. Simply enter the URL to your wishlist and all your wishes will be displayed.


== Installation ==

1. Upload the ZIP by using the Plugins menu
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate the widget in the Plguins menu in WordPress
4. Edit the widget settings to enter the URL to your wishlist

If you want to embed your wishlist in a blog post or page just use the shortcode, e.g.
[wishlist listurl="http://www.wunsch-index.de/wunschliste_von_max-1-aaaaaaaa.htm"]

The shortcode will be replaced with a listing of your wishes.
